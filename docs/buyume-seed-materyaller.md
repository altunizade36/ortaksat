## 1. Hazır başlık kalıpları (kategori bazlı)

**Kadın Giyim**
- `[Kumaş] [Kesim] [Ürün] - [Renk]` → "Saten Midi Abiye Elbise - Bordo"
- `[Sezon/Kullanım] [Ürün] - [Beden aralığı]` → "Yazlık Keten Gömlek - S-XL"
- `[Öne çıkan özellik] [Ürün]` → "Terletmeyen Bambu Bluz - 4 Renk"

**Takı & Mücevher**
- `[Ayar/Malzeme] [Motif] [Ürün] - [Renk/Kaplama]` → "925 Gümüş Kalp Kolye - Rose"
- `[Ürün] Set - [Adet] Parça` → "Zirkon Taşlı Küpe Set - 3 Parça"
- `Hediyelik [Ürün] - Kutulu` → "Hediyelik İsimli Kolye - Kutulu"

**Çanta / Ayakkabı**
- `[Malzeme] [Tip] [Ürün] - [Renk]` → "Hakiki Deri Omuz Çantası - Taba"
- `[Kullanım] [Ürün] - [Topuk/Boy]` → "Günlük Babet Ayakkabı - Düz Taban"

**Kural:** marka/model belliyse başa yaz; değilse ürün-tipi + en çekici özellik. Başlıkta ABARTMA/emoji yok.


## 3. DM varyantları — ORTAK daveti (3 sıcaklık düzeyi)

**Soğuk (hiç tanımıyorsun, IG/FB grubu)**
```
Selam 👋 Profilinde [moda/takı] paylaşımlarını gördüm. Ek gelir için ürün
paylaşmaya açık mısın? Stok/para yatırma yok — sadece linkini paylaşıyorsun,
satışta %15-20 komisyon. İstersen 1 dk'da anlatayım.
```
**Ilık (etkileşim var / arkadaş tavsiyesi)**
```
Merhaba [isim] 🙂 [ortak arkadaş/gruptan] senden bahsetti. OrtakSat'ta
[niş] ürünlerini paylaşıp komisyon kazanan bir ekip kuruyoruz. Senin
[kitlen/takipçin] tam uygun. Başlangıç linkini atayım mı?
```
**Takip (yanıt vermedi, 3 gün sonra — 1 kez)**
```
Selam, geçen mesajım kaçmış olabilir 🙂 Kısaca: paylaş → satışta komisyon,
başka yük yok. İlgilenirsen "olur" yaz, gerisini ben hallederim.
```
**Onay geldi → başlatma**
```
Harika! 3 adım:
1) ortaksat.com/partner → beğendiğin ürüne "Ortak ol"
2) Çıkan linki Instagram bio'na / WhatsApp durumuna koy (gönderiye link koyma, IG'de tıklanmaz)
3) Satışta komisyon otomatik takipte. İlk satışında yanındayım.
```

## 6. İlk-10-satış concierge checklist (her satış için)

- [ ] Ortak linki paylaştı mı? (bio/durum kontrol)
- [ ] Alıcı `/i/[slug]` landing'e geldi mi? (referral_clicks + channel'a bak)
- [ ] Talep/lead düştü mü? → satıcıya haber
- [ ] Satıcı satışı kaydetti mi? (`/seller` → Satış ekle)
- [ ] **Onay linki alıcıya gönderildi mi?** (kayıt anında çıkan "Onay linkini gönder" — EN KRİTİK)
- [ ] Alıcı `/onay/[token]`'da onayladı mı? → komisyon doğrulandı
- [ ] Satıcı komisyonu ödedi + "Ödedim" işaretledi mi?
- [ ] Ortak "Aldım" onayladı mı? → **kapandı = 1 kanıtlı satış**
- [ ] İzinle sosyal kanıt: "[Ortak] ilk ayında [X] ₺ kazandı" → sonraki DM'lere ekle
